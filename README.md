# MongoLib

MongoDB支持库

支持Bukkit, Spigot, BungeeCord, Sponge

插件本身无任何作用,只是作为前置插件

```xml
<repositories>
  <repository>
    <id>mcrmb</id>
    <url>http://repo.mcrmb.com/nexus/content/repositories/public/</url>
  </repository>
</repositories>

<dependencies>
  <dependency>
    <groupId>com.txgs888</groupId>
    <artifactId>MongoLib</artifactId>
    <version>3.4.2</version>
  </dependency>
</dependencies>
```
